# LiveKit Node reference

Current documented imports:

```ts
import { voice } from "@livekit/agents";
import * as lemonslice from "@livekit/agents-plugin-lemonslice";
```

Documented first-class inputs include `agentImageUrl`, `agentId`, `agentImage`, `agentImageMimeType`, `agentPrompt`, `agentIdlePrompt`, and `idleTimeout`.

Fields forwarded through `extraPayload` include `model`, `aspectRatio`, `responseDoneTimeout`, and `simulcast`.

Inspect installed package types/source because published package versions can lag documentation.
